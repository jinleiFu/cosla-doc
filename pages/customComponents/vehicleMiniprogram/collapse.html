# 折叠面板组件

# 说明

该组件需要搭配collapse-item一起使用，与uviewu-collapse的区别在于该组件中折叠面板项的高度不是通过动态计算得来的，同时为该组件添加了自定义样式。

# 代码示例

点击查看代码
<template>
	<view class="collapse-page">
		<nav-bar title="折叠面板组件" />
		
		<collapse accordion>
			<collapse-item title="所见">
				<view class="collapse-content">
					<view>牧童骑黄牛，歌声振林樾。</view>
					<view>意欲捕鸣蝉，忽然闭口立。</view>
				</view>
			</collapse-item>
			<collapse-item title="皇矣" :open="false">
				<view class="collapse-content">
					<view>皇矣上帝，临下有赫。监观四方，求民之莫。维此二国，其政不获。维彼四国，爰究爰度。上帝耆之，憎其式廓。乃眷西顾，此维与宅。</view>
					<view>作之屏之，其菑其翳。修之平之，其灌其栵。启之辟之，其柽其椐。攘之剔之，其檿其柘。帝迁明德，串夷载路。天立厥配，受命既固。</view>
					<view>帝省其山，柞棫斯拔，松柏斯兑。帝作邦作对，自大伯王季。维此王季，因心则友。则友其兄，则笃其庆，载锡之光。受禄无丧，奄有四方。</view>
					<view>维此王季，帝度其心。貊其德音，其德克明。克明克类，克长克君。王此大邦，克顺克比。比于文王，其德靡悔。既受帝祉，施于孙子。</view>
					<view>帝谓文王：无然畔援，无然歆羡，诞先登于岸。密人不恭，敢距大邦，侵阮徂共。王赫斯怒，爰整其旅，以按徂旅。以笃于周祜，以对于天下。</view>
					<view>依其在京，侵自阮疆。陟我高冈，无矢我陵。我陵我阿，无饮我泉，我泉我池。度其鲜原，居岐之阳，在渭之将。万邦之方，下民之王。</view>
					<view>帝谓文王：予怀明德，不大声以色，不长夏以革。不识不知，顺帝之则。帝谓文王：訽尔仇方，同尔弟兄。以尔钩援，与尔临冲，以伐崇墉。</view>
					<view>临冲闲闲，崇墉言言。执讯连连，攸馘安安。是类是禡，是致是附，四方以无侮。临冲茀茀，崇墉仡仡。是伐是肆，是绝是忽。四方以无拂。</view>
				</view>
			</collapse-item>
			<collapse-item title="望庐山瀑布" :open="false">
				<view class="collapse-content">
					<view>日照香炉生紫烟，遥看瀑布挂前川。</view>
					<view>飞流直下三千尺，疑是银河落九天。</view>
				</view>
			</collapse-item>
		</collapse>
		
	</view>
</template>

</script>
